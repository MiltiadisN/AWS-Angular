Welcome to my Angular App with AWS Integration!

This project showcases the integration of Angular with AWS services, including Lambda, API Gateway, and Cognito User Pool.

The authentication functionality is implemented using Cognito User Pool, allowing users to sign in, sign up, and perform other actions securely.

With the help of Lambda and API Gateway, the frontend seamlessly interacts with serverless backend services.

The user can view their hypothetical orders and the products they have ordered.
